# Truck image/report folders in this repo

These folders are **tracked in Git** and should appear at the **root** of the repo on GitHub:

- **TATA 1512G LPT-20260307T052513Z-1-001** – TATA 1512G LPT images and reports  
- **Tata Ace Gold (7908)-20260307T052536Z-1-001** – Tata Ace Gold (7908) images and reports  
- **Tata Ace Gold-20260307T052504Z-1-001** – Tata Ace Gold images and reports  

## Where to see them on GitHub

1. Open: **https://github.com/Pranshu115/vizon**
2. Make sure you’re on the **main** branch (dropdown at top left).
3. On the repo root page, scroll through the list of files and folders, or use **Ctrl+F** (Cmd+F on Mac) and search for:
   - `TATA 1512G LPT`
   - `Tata Ace Gold (7908)`
   - `Tata Ace Gold-20260307`

If you don’t see them:

- **Hard refresh:** Ctrl+Shift+R (Cmd+Shift+R on Mac), or try an incognito/private window.
- **Push again from your machine** (so GitHub has the latest):
  ```powershell
  cd "C:\Users\Administrator\Downloads\vizon-main\vizon-main"
  git push origin main
  ```
  If Git says the branch is behind the remote, run: `git push origin main --force` (this overwrites the remote with your local branch).
